
# Netflix Dataset Analysis

This repository contains an analysis of a Netflix CSV dataset, including both movies and TV shows. 
The dataset includes information such as titles, directors, cast, countries, release years, ratings, durations, and genres.

## Dataset
The dataset contains the following columns:
['show_id', 'type', 'title', 'director', 'cast', 'country', 'date_added', 
 'release_year', 'rating', 'duration', 'listed_in', 'description']

## Analysis
The analysis focuses on exploring and cleaning the data to gain insights. 
This includes handling missing values and duplicates, organizing and sorting entries, 
and examining patterns such as the number of movies and TV shows per director 
and the distribution of movie lengths. The goal is to provide a clear overview of 
the dataset while highlighting interesting trends and characteristics.
